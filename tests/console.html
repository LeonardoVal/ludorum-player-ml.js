Open your browser Javascript console. In Windows use: Ctrl+Shift+J in Chrome, Ctrl+Shift+K in Firefox, F12 in Internet Explorer, Ctrl+Shift+I in Opera.

ludorum_player_ml Automatic players based in statistical classification models for the Ludorum framework.
var MODEL = new ludorum_player_ml.examples.TicTacToeGameModel(); A game model provides the necessary metadata to process the state of a game with a classifier.
var CLASSIFIER_TYPE = ludorum_player_ml.LinearClassifier.actionClassifier(MODEL); A linear classifier selects a class for a game state based on a linear combination of its features. Action classifiers use the game possible actions (i.e. moves) as classes.
var CLASSIFIER = CLASSIFIER_TYPE.randomClassifier(); Randomly defined action linear classifier, obviously not expected to play well.
var PLAYER = new ludorum_player_ml.ActionClassifierPlayer({ classifier: CLASSIFIER }); An action classifier can be used to decide which action to take given a game state and a player. This is what the ActionClassifierPlayer does.
var MATCH = new ludorum.Match(GAME, [PLAYER, PLAYER]); Test match between players based on a randomly defined action classifier.
